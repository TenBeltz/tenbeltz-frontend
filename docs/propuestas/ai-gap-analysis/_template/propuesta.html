Objeto de la propuesta

El servicio `AI Gap Analysis` está planteado para ayudar a {{CLIENT_SHORT_NAME}} a tomar una decisión técnica clara antes de invertir tiempo en construir {{PROJECT_SUMMARY}}. El objetivo no es empezar a desarrollar todavía, sino definir si el caso tiene sentido, qué condiciones hacen falta y cuál sería la forma correcta de abordarlo.

Viabilidad técnica IA local Criterio de arquitectura Siguientes pasos

Alcance del servicio

01

Reunión de contexto

Sesión inicial para entender el proyecto, el contexto de negocio, el flujo de trabajo actual y el uso real que se quiere dar a la herramienta.

  • Recogida de información relevante para el análisis
  • Entendimiento del caso de uso y de las tareas a agilizar
  • Identificación de objetivos, restricciones y métricas deseables
02

Elaboración del informe

Tras la sesión de contexto, TenBeltz prepara un informe técnico con una propuesta razonada de enfoque.

  • Propuesta de plan y siguiente paso recomendado
  • Stack de herramientas sugerido
  • Análisis inicial de riesgos y previsión de complejidad
  • Metodologías recomendadas y no recomendadas para el caso
03

Reunión de presentación

Sesión final para presentar el informe, aclarar conclusiones y responder dudas antes de decidir el siguiente movimiento.

  • Presentación del análisis y de las recomendaciones
  • Aclaraciones sobre decisiones técnicas propuestas
  • Resolución de dudas y alineación sobre el siguiente paso
